základné príkazy:


ls - obsah priečinku
ls -l
ls -la
ls -lR
cd - change directory
cd .. - change directory do rodicovskeho subora
cd / - change directory na root
cd ~ - z root na domovsky priecinok
cd ~MENO - dostat sa do cudzieho priecinku
pwd - print working directory
whoami - kto ste
mkdir - make directory
rmdir - remove directory
